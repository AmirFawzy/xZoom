xZoom gallery with default options

All needed features are turned on by default, with adaptive and responsive support. Please, read the manual to see all options.

Mouse over the image, to see zoomed image window.

User mouse scroll to increase or decrease image in zoom window.

xZoom positioned window

Zoomed window can be positioned anywhere on page, just identify div with a custom name. Please, read the manual to see all options.

Mouse over the image, to see positioned zoomed image window.

xZoom lens option

You can change size, color, shape of lens. Please, read the manual to see all options.

Mouse over the image, to see zoomed image window.

Use mouse scroll to increase or decrease image in zoom window.

xZoom with FancyBox

Very useful feature for retail stores, FancyBox is working right from the box with xZoom gallry plugin, present all images to your customers! Please, read the manual.

Mouse over the image, to see zoomed image window.

Use mouse scroll to increase or decrease image in zoom window.

xZoom with Magnific Pop-Up

In this example we have tested undocumented yet new feature for mobile devices "fullscreen" mode, along with Magnific Pop-Up that also works right from the box with xZoom gallery plugin.

Tap to open "fullscreen" zoom on your mobile and double tap after that to open image with Magnific Pop-Up.

Mouse over the image, if you are not from mobile device to see zoomed image window.

Use mouse scroll to increase or decrease image in zoom window.

Please, read the manual for more options.

Features

Functionality

Lightweight code

Adaptive
